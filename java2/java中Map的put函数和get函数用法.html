    那么究竟put函数和get函数的用法是如何的呢？

    当然java中的Map集合是有Key和Value的。

     put（）函数

    V put(K key, V value)

    使用的参数：该方法有两个参数。

    •  key - 与指定值相关联的键。
    • value - 与指定键关联的值。

    返回值：当存在这个key的时候,会覆盖掉原来的value并返回oldvalue,也就是旧值。

    对返回值的进一步解释：

    1. 如果没有键映射，则返回NULL。
    2. 该函数返回与指定键关联的旧值。
    3. 这个操作不管啥条件都会覆盖旧的。
