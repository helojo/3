    jdk安装
    一、用yum安装jdk
    1.查看yum库都有哪些jdk版本
    yum search java|grep jdk
    2.选择版本安装
    yum install java-1.8.0-openjdk（/usr/lib/jvm/Java）
    3.设置环境变量
    vi  /etc/profile
    JAVA_HOME=/usr/lib/jvm/java-1.7.0-openjdk-1.7.0.75.x86_64
    JRE_HOME=$JAVA_HOME/jre
    CLASS_PATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar:$JRE_HOME/lib
    PATH=$PATH:$JAVA_HOME/bin:$JRE_HOME/bin
    export JAVA_HOME JRE_HOME CLASS_PATH PATH
    修改生效
    source /etc/profile
    二、手动解压方式
    1.创建Java目录
    mkdir/usr/java
    cd /usr/java
    2.下载解压
    curl -O http://download.oracle.com/otn-pub/java/jdk/7u79-b15/jdk-7u79-linux-x64.tar.gz
    tar -zxvf jdk-7u79-linux-x64.tar.gz

    Maven安装
    1.进入目录，并创建文件夹
    cd /usr/local
    mkdir maven
    2.下载并解压
    wget http://mirrors.hust.edu.cn/apache/maven/maven-3/3.5.4/binaries/apache-maven-3.5.4-bin.tar.gz
    tar -xvf apache-maven-3.5.4-bin.tar.gz 
    3.配置环境变量
    vim /etc/profile
    export MAVEN_HOME=/usr/local/maven/apache-maven-3.5.4
    PATH=$JAVA_HOME/bin:$MAVEN_HOME/bin:$PATH
    source /etc/profile
    Tomcat安装
    1.进入目录，并创建文件夹
    cd /usr/local
    mkdir tomcat
    2.下载并解压
    wget http://mirrors.tuna.tsinghua.edu.cn/apache/tomcat/tomcat-8/v8.5.4/bin/apache-tomcat-8.5.4.tar.gz
    tar -zxvf apache-tomcat-8.5.4.tar.gz 
    3.启动和关闭
    cd /usr/local/tomcat/bin
    ./startup.sh
    shutdown.sh
    4.配置web管理账号
    vim /usr/local/tomcat/conf/tomcat-users.xml
