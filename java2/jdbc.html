    eclipse下导入mysql驱动

    eclipse版本：    

     

    mysql版本：

    mysql Ver 8.0.16 for Win64 on x86_64 (MySQL Community Server - GPL)

    驱动版本：

    mysql-connector-java-8.0.16.jar

     

    步骤：

    1.在文件工程鼠标右键，找到Build Path ，选择Configure Build Path..

     

    2.点击后选择jar包，打开

     

     

    3.成功

     

     

     

    4.测试：

    （注意：mysql8.0版本以后Class.forName("com.mysql.jdbc.Driver");   语句加载驱动已经不适应，应该改为Class.forName("com.mysql.jdbc.Driver");）

    加载驱动代码：

    package jdbcTest;

    public class Test {

    public static void main(String[] args) {

    // TODO Auto-generated method stub

    try {

    Class.forName("com.mysql.cj.jdbc.Driver");

    } catch (ClassNotFoundException e) {

    // TODO Auto-generated catch block

    e.printStackTrace();

    }

    }

    }
