    1.http协议的组成
    http:规范那种协议
    localhost、127.0.0.1:访问的ip地址（默认，根据自己的需求改变）
    端口号：8080（默认，根据自己的需求改变）
    工程：XXX
    资源:可以是jsp，html,servlet 等等。
    http://localhost:8080/工程名/要访问的资源/
    http://localhost:8080/XXX/login.jsp

     如：
Request URL: http://localhost:8080/WebDemo/demo.html
Request Method: GET（请求方式）
Status Code: 304 （状态码）
Remote Address: [::1]:8080（http版本和端口号）
