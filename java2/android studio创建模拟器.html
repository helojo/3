    开发环境:

    • 操作系统: windows10 教育版 1903
    • Android studio :  Android Studio 3.5.1
      Build #AI-191.8026.42.35.5900203, built on September 26, 2019
      JRE: 1.8.0_202-release-1483-b03 amd64
      JVM: OpenJDK 64-Bit Server VM by JetBrains s.r.o
      Windows 10 10.0

    以下是详细步骤:

    • 打开 Android Studio 并点击 AVD Manager。

    •  点击《创建虚拟设备》。

    • 选择硬件类型并点击下一步, 这里我选择的是 5.0英寸的手机端设备。

    •  选择系统镜像并点击下一步。

    • 修改模拟器名称并点击【Finish】结束配置(如果有需要的话可以在这里修改其他配置)。
    • 完成。
    • 启动模拟器。点击该模拟器的【运行】按钮，运行模拟器。
