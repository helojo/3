    • Window中在Intellij idea开发时常用快捷键
    原文出处:本文由博客园博主AndyBBOY提供。
    原文连接:https://www.cnblogs.com/self-study/p/11692684.html

    以下idea中的快捷键是在window 7中确认过，如果快捷键不起作用，可能是该快捷键被其它软件占用，或系统不同导致。

    1、Ctrl + Z：撤回代码；

    2、Ctrl + Shift + Z：恢复撤回的代码；

    3、Ctrl + F：在类文件中查找某个变量名；

    4、Ctrl + D：复制当前行代码到下一行；

    5、Ctrl + Y：删除当前行代码；

    6、 Ctrl + N：查找工程中的类； Ctrl + Shift + N：查找工程中非Java文件；

    7、Ctrl + Shift + F/R：在工程中搜索文件（自己写的）；双击Shift：搜索工程中所有类文件，包括引用的jar包中的；

    8、Alt + Insert：创建类中字段的getter与setter的方法；

    9、Shift + F6：选中目录结构的文件重命名；

    10、Alt + F1：回到当前文件所在的目录结构；

    11、Ctrl + E：最近打开的文件；

    12、Ctrl + P：方法参数提示；

    13、Ctrl + Alt + F12：进入当前文件所在的磁盘位置；

    14、Ctrl + Shift + C：复制当前文件磁盘路径到剪贴板；

    15、Ctrl + O：重写方法；

    16、Ctrl + I：实现方法；

    17、Ctrl + R：当前文件变量替换；

    18、Ctrl + Shift + L：格式化代码；

    19、Ctrl + Shift + F12：显示代码的窗口最大化，再次点击恢复原状；

    20、Ctrl + Alt + B（或左方向键、或鼠标左键）：进入函数或接口的实现类；

    21、Ctrl + Shift + U：大小写切换；

    22、Ctrl + Shift + Enter：对当前代码行末尾添加分号；

    23、Alt + 7：查看类中有哪些方法；

    24、Ctrl + Alt + F7：查看定义的变量在哪里被调用；

    25、Ctrl + H：查看当前类的继承树；

    26、 itar ：生成array for代码块；

    27、iter ：生成增强for；

    28、itli ：生成List的遍历；

    29、itit： 生成iterator 迭代；

    30、Ctrl + Alt + T：常用的一些方法（条件、循环、try-catch等）；

    31、psvm：快速写mian函数；

    32、sout：快速写出输出打印System.out.println();
